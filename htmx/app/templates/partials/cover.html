Solana DeFi Ecosystem Dashboard

This dashboard monitors the ONyc ecosystem across DEX, lending, and yield venues on Solana. It is designed to surface token issuance, liquidity quality, leverage, yield-market structure, and downside risk in one place, using a live data pipeline that continuously decodes on-chain transactions and account state changes into time-series views refreshed every few seconds.

Use the View selector in the header to navigate between ecosystem, protocol, risk, and system-health pages. The Last dropdown controls the lookback window for every chart on the current page — from 1 hour to 90 days. Most charts have an expand button (↗) in the lower-right corner for a full-screen view.

ONyc Token Ecosystem

What is Monitored

ONyc is a Solana SPL token active across DEX, lending, and yield-tokenisation venues. This dashboard tracks its issuance, deployment, trading, lending, and transformation in real time.

  • Token issuance — ONyc, SY, PT, and YT supply across monitored venues
  • DEX liquidity — depth, concentration, LP structure, and sell-side price impact
  • Lending markets — reserve utilisation, borrow demand, collateral mix, and borrower health
  • Yield markets — implied fixed rates, AMM capital, escrow balances, and maturity-pair activity
  • Ecosystem structure — supply deployment, TVL, and activity across monitored venues

Key Activities Tracked

The pipeline captures the core on-chain event types that drive market structure, liquidity, credit exposure, and yield formation across the ONyc ecosystem:

  • Swap events — buy / sell flow, trade sizing, and downside move analysis on Orca & Raydium
  • Liquidity events — deposits, withdrawals, LP position changes, and depth reconfiguration
  • Lending events — supply, borrow, repay, withdrawal, and liquidation activity on Kamino
  • Yield events — SY / PT / YT issuance, redemption, trading, and AMM inventory changes on Exponent

Cross-Protocol Analysis

Beyond per-protocol monitoring, the dashboard integrates data across venues to surface ecosystem-level patterns and cross-protocol risk transmission:

  • Supply deployment — ONyc distribution across DEX liquidity, lending reserves, and yield markets
  • Downside move risk — liquidity depth exhaustion and move-probability estimates under sell pressure
  • Liquidation exposure — debt-at-risk and borrower sensitivity to collateral or borrow-asset shocks
  • Yield comparison — base token yield, lending APY, and implied fixed rates across maturity markets
  • Cascade analysis — linking lending stress scenarios to DEX liquidity impact and cross-venue fragility

How It Works

Pipeline Overview

1

Ingest

Live Solana event streams and periodic account snapshots capture the on-chain transactions and state updates that matter for market activity, reserve balances, and position health.

2

Decode & Enrich

Raw protocol data is decoded, normalised, and matched to pool, reserve, market, and token metadata so activity from different venues can be analysed on a common basis.

3

Store & Compute

Events are stored in a time-series PostgreSQL / Timescale layer where much of the analytics engineering is performed in-database using SQL functions, reusable abstractions, and continuous aggregates designed for efficient time-series analysis.

4

Serve

A lightweight API serves parameterised chart queries to the dashboard, with automatic refresh and fast page loads across ecosystem, protocol, risk, and system-health views.

Data Notes

Historical Coverage

  • Live coverage — continuous event and balance monitoring began on 16 March 2026. Data from this date forward is captured directly by the live pipeline.
  • Earlier history — balances prior to 16 March 2026 have been reconstructed from transaction history. This backfill has been extensively validated, but small differences versus a long-running production indexer may still occur.
  • Kamino obligation history — position-level obligation balances on Kamino are only available from the live-ingestion start date onward. Earlier Kamino history should therefore be interpreted at reserve / market level rather than individual obligation level.
