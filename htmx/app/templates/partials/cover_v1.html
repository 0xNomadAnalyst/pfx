Getting Started

Navigate Pages

Use the View selector in the header bar to switch between dashboard pages. Each page focuses on a different protocol or cross-protocol perspective.

Time Windows

The Last dropdown controls the lookback period for every chart on the current page — from 1 hour to 90 days. Data refreshes automatically every 15–90 seconds.

Expand & Explore

Most charts have an expand button (⤢) in the lower-right corner. Click it to open a full-screen view for closer analysis.

ONyc Token Ecosystem

ONyc is a Solana SPL token whose DeFi footprint spans decentralised exchanges, lending markets, and yield tokenisation protocols. This dashboard tracks every on-chain interaction across those venues in real time, providing a single-pane view of supply distribution, capital utilisation, and protocol activity.

What Is Monitored

  • Token supply and derivative issuance (SY, PT, YT)
  • Liquidity depth and LP positions across DEX pools
  • Lending supply, borrow rates, and collateral utilisation
  • Yield curves: base token yield, implied APY, lending APY
  • Protocol-level TVL and 24h activity volumes

Key Interactions Tracked

  • Swap events — buy / sell volumes on Orca & Raydium
  • Liquidity events — deposits, withdrawals, position changes
  • Lending events — supply, borrow, repay, liquidation
  • Yield events — PT/YT mints, redeems, and trades

How It Works

1

Ingest

Real-time gRPC streams (Yellowstone) plus periodic RPC snapshots capture every relevant on-chain event and account state change.

2

Decode & Enrich

Raw transactions are Borsh-decoded, matched against protocol discriminators, and enriched with pool/market metadata before persisting.

3

Store & Roll Up

Events land in TimescaleDB hypertables partitioned by time. Continuous aggregates refresh every 5 seconds for sub-minute dashboard freshness.

4

Serve

Parameterised view functions feed a lightweight API layer. The dashboard fetches each widget independently via HTMX with staggered loads and automatic refresh.
